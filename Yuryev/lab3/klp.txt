ab//b//

Дополнительные примеры, чтобы ускорить проверку
Example 1: (полное 4-уровневое дерево)
abcd//d//cd//d//bcd//d//cd//d//

Example 2: (большое кол-во уровней по "случайным" сторонам)
abc//c//bcd///cde///d/e/fgh////

Example 3: (большое кол-во уровней по одной стороне дерева)
a/b/c/d/e/f/g/h/i/j/k/l/m/n/o/p//

Example 4: (дерево состоит из одного корня)
a//

Example 5: (пустое дерево)
/

Example 6: (чрезмерное количество символов "пустоты")
ab/////////////////

Example 7: (большое количество символов табуляции и пробела между значащими символами)
a       b       /       /       b       /       /

Example 8: (ввод каждого значащего символа после символа \n (перевод на новую строку))
a
/
b
/
/

Example 9: (ввод цифр)
12//2//
