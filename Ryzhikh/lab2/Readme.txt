При запуске программы без аргументов, программа считывает бинарное коромысло с консоли.
При запуске с аргументом, программа считывает из файла, название которого - переданный аргумент.
В input.txt лежит пример вводимых программе данных.
В link.txt лежит ссылка на олнайн-компилятор, в котором можно запустить код.
Дополнительно ничего устанавливать не нужно.
