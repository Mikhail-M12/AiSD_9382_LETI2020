Инструкция по пользованию программой.

При запуске программа просит пользователя ввести режим ввода(0 - ввод из файла tests.txt, 1 - ввод через консоль).
При выборе ввода через консоль, пользователь должен ввести сначала количество элементов во входном массиве, а затем - сами элементы, разделяя их пробелом.
После этого программа выведет:
1) Часть массива, обрабатываемую очередным вызовом функции quickSort()
2) Границы этой части массива(индексирование элементов начинается с 0), а также опорный элемент
3) Состояние массива на каждой итерации внешнего цикла for в функции partition()
4) После того, как будут завершены вызовы всех quickSort() - отсортированный в порядке возрастания массив

Элементы массива на каждом шаге разделены на 3 области(они схематично "отгорожены" вертикальными чертами) в следующем порядке:
1) Элементы < опорного
2) Опорный элемент и элементы >= опорного
3) Ещё неотсортированные элементы

Запустить программу можно в онлайн-компиляторе www.onlinegdb.com/online_c++_compiler
