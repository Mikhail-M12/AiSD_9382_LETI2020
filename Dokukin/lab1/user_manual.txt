При запуске программа просит пользователя выбрать режим ввода.
Введите 0, чтобы загрузить данные из файла tests.txt, или 1, чтобы ввести свою строку.
После выбора режима программы последовательно выведет на экран:
1. Введённую строку
2. Значения переменных dotpos и epos - позиции символов "." и "Е" соответственно
3. Значения строк str1, str2, str3, на которые разбивается исходная
4. Ход рекурсии, указывая на каждом шаге глубину и обрабатываемый символ
5. Результат обработки

Для запуска программы можно использовать онлайн-компилятор www.onlinegdb.com/online_c++_compiler
